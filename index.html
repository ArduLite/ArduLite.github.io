Library/framework ringan untuk C++ yang dirancang khusus untuk board Arduino dengan memori terbatas.

Overview

ArduLite adalah library ringan yang dirancang untuk menyediakan fungsionalitas penting untuk Arduino dengan footprint minimal.

ArduLite sangat ideal untuk proyek dengan sumber daya terbatas di mana efisiensi dan kesederhanaan sangat penting.

Board yang didukung:

  • Arduino Uno
  • Arduino Nano
  • Board berbasis ATMega8

Tujuan utama dari ArduLite adalah menghemat penggunaan memori dibandingkan dengan framework Arduino standar, sehingga memungkinkan aplikasi berjalan lebih efisien di perangkat dengan sumber daya terbatas.

Fitur ArduLite

Berbagai fitur yang tersedia di ArduLite.

Penanganan Pin Digital & Analog

Kontrol mudah untuk pin digital dan analog dengan sintaks yang disederhanakan.

Dukungan PWM

Kontrol perangkat dengan modulasi lebar pulsa (PWM) untuk berbagai aplikasi.

Sintaks yang Disederhanakan

Sintaks yang mudah dipahami dan digunakan, cocok untuk pemula maupun ahli.

Optimasi Penggunaan Memori

Dioptimalkan untuk penggunaan memori minimal, cocok untuk perangkat dengan memori terbatas.

Mengapa ArduLite?

Keunggulan menggunakan ArduLite untuk proyek Arduino Anda.

  • Lebih Hemat Memori - Program "blink" standar pada Arduino memakan 924 bytes, sementara dengan ArduLite hanya 190 bytes saja!
  • Desain Kode Lebih Elegan - Struktur kode tetap mirip dengan Arduino asli, sehingga tetap mudah dipahami.
  • Cocok untuk Board Low-Memory - Dirancang khusus untuk board dengan SRAM & Flash kecil.
  • Mudah Dipelajari - Sintaks yang sederhana membuatnya mudah dipelajari oleh pemula.

Cara Instalasi

Panduan lengkap untuk menginstal ArduLite di Arduino IDE Anda.

Metode 1: Melalui Library Manager Arduino IDE:

  1. Buka Arduino IDE
  2. Masuk ke menu Sketch > Include Library > Manage Libraries
  3. Cari ArduLite
  4. Klik Install

Metode 2: Install Manual dari GitHub:

Download release terbaru dari GitHub atau clone repository dengan perintah:

git clone https://github.com/ArduLite.git

Ekstrak folder dan letakkan di dalam direktori libraries Arduino.

Restart Arduino IDE.

Sertakan library di proyek Anda:

#include <ArduLite.h>

Penggunaan

Contoh penggunaan ArduLite dalam berbagai skenario.

Kontrol Digital Dasar

Contoh membaca tombol dan mengontrol LED:

#include <ArduLite.h>

int main() {
  Digital button(2, IN);
  Digital led(13, OUT);

  while (1) {
    if (!button.read()) {
      led.toggle();
      wait(500);
    }
  }
}

Kontrol PWM

Contoh mengatur kecerahan LED dengan PWM:

#include <ArduLite.h>

int main() {
  PWM led(9);

  while (1) {
    for (byte i = 0; i < 255; i++) {
      led.write(i);
      wait(5);
    }
    for (byte i = 255; i > 0; i--) {
      led.write(i);
      wait(5);
    }
  }
}

Membaca Sensor Analog

Contoh membaca sensor analog dan mengontrol kecerahan LED:

#include <ArduLite.h>

int main() {
  PWM led(9);
  Analog sensor(A0);

  while (1) {
    int sensorValue = sensor.read();
    float brightness = sensorValue * (255.0 / 1023.0);
    led.write(brightness);
  }
}

ArduLite

üé® ArduLite - Lite Version of Arduino ‚ö°Ô∏è

Lightweight C++ library/framework specifically designed for Arduino boards with limited memory.

Overview

ArduLite is a lightweight library designed to provide essential functionalities for Arduino with minimal footprint.

It is ideal for resource-constrained projects where efficiency and simplicity are crucial.

Supported boards:

  • Arduino Uno
  • Arduino Nano
  • Boards based on ATMega8

The main goal of ArduLite is to save memory usage compared to the standard Arduino framework, allowing applications to run more efficiently on devices with limited resources.

ArduLite Features

Various features available in ArduLite.

Digital & Analog Pin Handling

Easy control for digital and analog pins with simplified syntax.

PWM Support

Control devices with pulse width modulation (PWM) for various applications.

Simplified Syntax

Easy-to-understand and use syntax, suitable for both beginners and experts.

Optimized Memory Usage

Optimized for minimal memory usage, suitable for memory-constrained devices.

Why ArduLite?

Advantages of using ArduLite for your Arduino projects.

  • More Memory Efficient - Standard "blink" program on Arduino takes 924 bytes, while with ArduLite it's only 190 bytes!
  • More Elegant Code Design - Code structure remains similar to original Arduino, making it easy to understand.
  • Suitable for Low-Memory Boards - Specifically designed for boards with small SRAM & Flash.
  • Easy to Learn - Simple syntax makes it easy for beginners to learn.

Installation Guide

Complete guide to install ArduLite in your Arduino IDE.

Method 1: Via Arduino IDE Library Manager:

  1. Open Arduino IDE
  2. Go to Sketch > Include Library > Manage Libraries
  3. Search for ArduLite
  4. Click Install

Method 2: Manual Installation from GitHub:

Download the latest release from GitHub or clone the repository with command:

git clone https://github.com/ArduLite.git

Extract the folder and place it inside the Arduino libraries directory.

Restart the Arduino IDE.

Include the library in your project:

#include <ArduLite.h>
