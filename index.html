Tentang ArduLite

ArduLite adalah library/framework ringan untuk C++ yang dirancang khusus untuk board Arduino dengan memori terbatas.

Board yang didukung:

  • Arduino Uno
  • Arduino Nano
  • Board berbasis ATMega8

Tujuan utama dari ArduLite adalah menghemat penggunaan memori dibandingkan dengan framework Arduino standar, sehingga memungkinkan aplikasi berjalan lebih efisien di perangkat dengan sumber daya terbatas.

Mengapa ArduLite?

Keunggulan menggunakan ArduLite untuk proyek Arduino Anda.

  • Lebih Hemat Memori - Program "blink" standar pada Arduino memakan 924 bytes, sementara dengan ArduLite hanya 190 bytes saja!
  • Desain Kode Lebih Elegan - Struktur kode tetap mirip dengan Arduino asli, sehingga tetap mudah dipahami.
  • Cocok untuk Board Low-Memory - Dirancang khusus untuk board dengan SRAM & Flash kecil.

Perbandingan Penggunaan Memori

Library Penggunaan Flash Memory
Arduino Standar 924 bytes
ArduLite 190 bytes

Dengan ArduLite, Anda bisa menghemat lebih dari 79% memori!

Cara Instalasi

Panduan lengkap untuk menginstal ArduLite di Arduino IDE Anda.

Melalui Library Manager Arduino IDE:

  1. Buka Arduino IDE
  2. Masuk ke menu Sketch > Include Library > Manage Libraries
  3. Cari ArduLite
  4. Klik Install

Alternatif: Install Manual dari GitHub:

Clone repository dengan perintah:

Terminal 
git clone https://github.com/ArduLite.git

Tambahkan ke Arduino IDE:

  1. Buka Arduino IDE
  2. Masuk ke menu Sketch > Include Library > Add .ZIP Library
  3. Pilih folder ArduLite

Gunakan di Sketch Arduino:

C++ 
#include "ArduLite.h"

int main() {
    Digital led(13, OUT);

    while(1){
        led.on();
        wait(300);
        led.off();
        wait(300);
    }
}

About ArduLite

ArduLite is a lightweight C++ library/framework specifically designed for Arduino boards with limited memory.

Supported boards:

  • Arduino Uno
  • Arduino Nano
  • Boards based on ATMega8

The main goal of ArduLite is to save memory usage compared to the standard Arduino framework, allowing applications to run more efficiently on devices with limited resources.

Why ArduLite?

Advantages of using ArduLite for your Arduino projects.

  • More Memory Efficient - Standard "blink" program on Arduino takes 924 bytes, while with ArduLite it's only 190 bytes!
  • More Elegant Code Design - Code structure remains similar to original Arduino, making it easy to understand.
  • Suitable for Low-Memory Boards - Specifically designed for boards with small SRAM & Flash.

Memory Usage Comparison

Library Flash Memory Usage
Standard Arduino 924 bytes
ArduLite 190 bytes

With ArduLite, you can save more than 79% of memory!

Installation Guide

Complete guide to install ArduLite in your Arduino IDE.

Via Arduino IDE Library Manager:

  1. Open Arduino IDE
  2. Go to Sketch > Include Library > Manage Libraries
  3. Search for ArduLite
  4. Click Install

Alternative: Manual Installation from GitHub:

Clone the repository with command:

Terminal 
git clone https://github.com/ArduLite.git

Add to Arduino IDE:

  1. Open Arduino IDE
  2. Go to Sketch > Include Library > Add .ZIP Library
  3. Select the ArduLite folder

Use in Arduino Sketch:

C++ 
#include "ArduLite.h"

int main() {
    Digital led(13, OUT);

    while(1){
        led.on();
        wait(300);
        led.off();
        wait(300);
    }
}
